# Rubric — Review

Same baseline: code E (+ PRD A + RFC C). Score **1–10** each dimension + essay.

| # | Dimension | Low (1–3) | Mid (4–7) | High (8–10) |
|---|-----------|-----------|-----------|-------------|
| 1 | Bug finding | Misses real issues / only nits | Some real bugs | Finds meaningful defects |
| 2 | Severity calibration | Everything P0 or everything nit | Roughly right | Clear P0/P1/P2 |
| 3 | Spec grounding | Untethered opinions | Occasional PRD/RFC links | Findings tied to PRD/RFC |
| 4 | Actionability | Vague “improve this” | Partial guidance | Concrete fixes |
| 5 | False-positive control | Invents problems | Some noise | High signal; few invented issues |

**Essay:** Good or bad review? Why? (Blind.)
