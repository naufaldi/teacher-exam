# Rubric — RFC

Same baseline PRD A for both candidates. Score **1–10** each dimension + essay.

| # | Dimension | Low (1–3) | Mid (4–7) | High (8–10) |
|---|-----------|-----------|-----------|-------------|
| 1 | Spec fidelity | Ignores or rewrites PRD | Mostly follows; silent extras | Faithful to PRD A; conflicts called out |
| 2 | Architecture fit | Wrong stack / greenfield | Partial seams | Correct shared schema, API layers, web routes |
| 3 | Edge cases | Happy path only | Some gates | Plan gates, races, idempotency/webhooks if claimed |
| 4 | Phasing | Big-bang | Vague phases | Clear MVP slices + non-goals |
| 5 | Implementability | Coder must guess | Gaps remain | Executable without inventing major decisions |

**Essay:** Good or bad RFC? Why? (Blind.)
